Reigns 攻略小记


Reigns

如果你搜到了这篇文章，这证明你或许正被这个游戏所困惑，所以，这也正是这篇文章的本 意。

经过我的不懈努力，终于将这个游戏玩到了全成就。不过我的 Steam 截图出了点问题，貌 似只能截三张，略尴尬。

我玩的是在 Steam 上最后更新时间为 <2017-03-29 三> 的版本，希望对你能有所帮助。

Tips: 用电脑玩可以使用键盘左右键和空格进行操作，如果懒得动鼠标的话。

首先，我玩这个游戏的流程如下。

遇到幽灵 -> 招募医生 -> 遇到女巫 -> 第一次遇到恶魔在 666 年 ->

地下城之迷失 -> 和骷髅对话 -> 和瓶子对话 -> 发动圣战 -> 结束圣战 ->

向女巫学习骗过恶魔的方法（非必需） -> 女巫送礼 -> 狗触发红宝石（非必需） -> 霜冻之血（非必需） ->

冻结之血 -> 第二次遇到恶魔在 1332 年 -> 发动圣战 -> 拉马士革秘密 -> 获得草莓 ->

去地下城找骷髅 -> ["骗过恶魔 The End n°3", "对幽灵点头 The End n°2", "随便 杀死一个人 The End n°1"]

一开场必然会遇到幽灵，他会问你几个无聊的问题，但这不重要，重要的是他会解锁 The End n°2 成就，只要你在 1998 年遇到恶魔，获得杀人的叉子，之后他很可能会 出现，对他点头，国王就会腐烂在永恒的地狱，解锁成就，然后开始新游戏。这是第二个 结局。

招募医生和遇到女巫，遇到恶魔都是正常的剧情，同样不重要，重要的是 迷失在地下 城 。在地下城可以做两件事情，同样都需要 找到火把 这个先决条件，但并不需要从 女巫口中得知骗过恶魔的方法，没有那串数字也可以和骷髅对话。

  1. 第一件事是沿着 土，火，水，酸，宝剑之门 的顺序走，拿到宝剑，宝剑可以屠龙， 但你也会死，不过死亡方式会变成 "您用石中剑杀死了巨龙，但却在搏斗中受了重伤， 您像英雄一般被厚葬"，可惜这和统治时间过长，名字被人铭记的那个死亡方式一样， 没什么不同。（这把剑其实名叫亚瑟王神剑，可能会给游戏带来好运气。）
  2. 第二件事是和骷髅对话，我前面说过，女巫告诉你骗过恶魔的方法并不重要（事实上 女巫惟一的用处我认为就是让你和鸽子结婚），没有 1043，683，1214，842，1432， 1669 这串数字依然可以和骷髅对话，因为这串数字是写死在游戏里的，截止目前更 新的所有版本都是这串数字，所以很简单，我们只需要沿着 火，砷，酸 的顺序走， 见到 黄金之门不要进，不要进，不要进！第一次见到正五芒星也不要进，不要进， 不要进！ 之后就可以愉快的进入第二个正五芒星之门，和可爱的小骷髅对话了。在 进地下城的时候不要着急，只要看到 砷之门 - 盐之门 一定要选 盐之门 ，不然 会死。

之后你会从医生那里触发瓶中小人的剧情，也就是让你和瓶子对话，但这不重要，期间可 能会碰到发动圣战的剧情，发动之后再结束就可以了，没有任何影响。

随着瓶中小人的剧情结束，会让你寻找 冻结之血 ，期间可能会向女巫学习骗过恶魔的 方法，结束后很可能小狗会跑过来给你一颗红宝石，随即触发 霜冻之血 ，但要注意， 这是伪剧情，不一定会触发，据我所知， 霜冻之血 没有任何用处，只是为了让你将它 和 冻结之血 搞混。

所谓的女巫送礼才是真正的剧情，她会告诉你在老眼昏花的情况下执政十五年，结束后给 你皇冠或者红宝石作为奖励，注意，一定要拿 红宝石 ，之后触发 冻结之血 。

随着时间的推进，我们会在 1332 遇到恶魔，但这依然不重要，它可能只是在提示你，该 去拿草莓了。此时此刻，如果你有机会，那就发动圣战吧，在圣战中努力寻找圣杯，试图 触发 拉马士革的秘密 这一剧情，在剧情中从瓶子手里获得 草莓 。终于拿到了草莓， 或许可以常舒一口气，因为这已经离结束近在咫尺。（存在另一个剧情，黑皮肤黄衣服的 公主会告诉你拉马世革的秘密，它告诉你跟我来吧，随即触发草莓剧情）

手里有了草莓，我们自可以悠悠自在，在适当的时机，进地下城即可，进去的方法不一， 比较隐秘的方法就是教会达到 100，或者对狗不断的摇头，直至带入地下城。进入地下城 后，重复上述和骷髅对话的步骤，将草莓给他，他会和你一起出去，并在某项数值较低的 时候，给予你帮助，可惜的是，这依然不是最重要的。

最重要的时刻即将来临，这也是让你获得三次不同成就，体验三次死亡方式，外加一次 True pacifist 成就（当然，这些成就并不是一次完成）。

1998，恶魔最后一次降临于此，赐予你用叉子斩杀任何一人的机会。

  1. 机会一，用叉子随便杀一人，解锁 The End n°1 ，被困在无尽的中世纪。
  2. 机会二，当游戏最初的那个幽灵和你对话之时，对他点头，解锁 The End n°2 。
  3. 机会三，憋住叉子不用，一直死，直至骷髅出现，对他点头，解锁 The End n°3 ，真正骗过机会二的恶魔。
  4. 机会四，依旧是憋住叉子不用，不论是幽灵，骷髅，一百年之后，解锁 True pacifist 。

成就

至此，主线任务结束，不出意外的话，你已经获得了， Shooting star ， Elder ， Millenium ， All the characters ， The End n°1 2 3 ， Duelist ， True pacifist ， Hecatomb 这些成就。

剩下的比较坑爹的一些成就基本就是 Mummy ， All the cards ， All the deaths 这几个。

Mummy 执政超过百年其实没什么好说的，基本就是靠对游戏的熟悉程度，甚至是一点点 运气的成分，当然你最好一直对狗摇头，进丛林找到查看数值的眼珠，这样时间久了，你 也会记住一些回答对于数值的影响，毕竟它们都是一些固定值。

All the cards 这个成就我这边解锁的比较匪夷所思，玩着玩着就突然解锁了，但记得 最后遇到的是不会说人话的外国国王，所以可能对你没有太大的帮助。

All the deaths 当然也是最困难的一个，究其原因很可能是 Reigns 新增了一个极其 诡异名为红蚂蚁的死亡方式。除红蚂蚁外，可以在 Steam 指南中看一下这篇文章（可能 以后会更新到红蚂蚁），不论如何，我还是先来说一说我遇到的一些奇葩死亡方式。

  1. 被人贯穿眼珠射死

    这个貌似是新增的，触发条件应该是国库 100，人民或者军队很高。

  2. 心脏病 Heart Attack

    这个真的是太坑了，可以看到我为了得到这个死法，直接玩到了 2579 年。并且上面 那个 1944 年也是为了这了这个死法。触发方式为找到一位恋人，之后国库达到 100， 但这貌似是非标准死亡，指南里没有说明存在什么标准死亡方式。之前运气好的话， 可以让国库自动加钱，这样会简单很多，只要坐着不动自动跑到 100 即可，只要别让 教会变为 0。需要注意的是，不建议和鸽子结婚这种奇葩方式，因为我每次都和鸽子 结婚，结果国库达到 100 也没有触发 = =

  3. 被毒死 Espionage

    记住这个手持正义之手的人，他在剧情中扮演国王的兄弟，碰到他后，一旦你的军队 数值为 0，就会被他毒死。

  4. 死后被崇拜 Reverence

    上面提到过，用宝剑可以杀龙。但死法和统治时间过长自然死亡一样。

  5. 神秘杀手的斩首 Beheading

    这个我记不太清，触发之一应该是将军在地牢发现一具未知的尸体，问你需要去调查 吗？此时选择摇头，然后等待时机令军队达到 100。

  6. 被儿子杀死或者十年无聊中一直选左边 Adventures

    游戏会问你是否要处理私生子问题，留他一命，你会获得继承人，之后这个逆子要杀 死所有异教徒，调查他，很可能触发这个剧情，决斗当中输给他即可。

  7. 弑父 Patricide

    当你见到了你的继承人之后，让军队达到 100，这个混帐东西最终会把你吊死。

  8. 共产主义好 Communism

    国库非常高，人民达到 0，死后看起来像是一个身穿粉色大裤衩的国王在举重。😂

  9. 被暴民扔出窗户 Gravity

    建立殖民地后，令人民达到 100。

  10. 红蚁 Red ants

    这就是传说中新增的红蚂蚁剧情，对我来说也是最匪夷所思的一个触发方式，我无法 保证我的方法一定可行，但至少我是遇到了这个死法三次有余。第一种方法是，如果 你手里有宝剑和草莓，遇到医生后，同意他的某些肮脏研究，随即立刻死亡，被红蚁 吞噬。第二种方法，资助那个绿色背景名为 杰斯特・科特内 的人，他会发现美丽 的新世界（我不确定宝剑是否为必须之物，但我触发很多事件之时，身上都有它的存 在）。

    发现新大陆后，选择建立殖民地，找到机会，发动圣战！之后很可能会出现这个 帕 克勋爵 ，他会引荐一位女性朋友 麦伊莎公主 给你，注意，就是这位 麦伊莎公 主 ，触发红蚂蚁的关键人物，对着她乱扯一通，其后半载，红蚁吞噬，王猝（如果 帕克勋爵 不出现，将会是 麦伊莎公主 直接和你对话）。

其他

获得叉子后，其实对瓶子也可以点头，当他问你是否想要更改之前的抉择之时，只不过 过一会那个烦人的医生又把瓶子带了回来。

不要顺着水跑，尽量往右跑。

找到草莓之前在地下城见骷髅多少次都没有用，最后还是走出门。手里有宝剑的话再去 宝剑之门同理。

与小丑玩红矮人可以赌很多很多次，没钱了也可以，选择 "由我来定数额"，赢钱后可以 加一倍，小心国库会爆到 100（这个烦人的小丑真是和第十二夜中的小丑费斯特差远了）。

小丑的礼物右手（你的左边）永远是黄金十字架。

虽说我没有用鸽子开出来 Heart Attack，但我真的是爱死那只小白鸟了，貌似一只是叫 咲夜，一只是叫皮咕？

遇到恶魔后会有黑暗陷阱，谁能告诉我这是什么，我记不得了。

殖民地中那个黑皮肤黄衣服的公主貌似是之前来的商人还试图劝过你停止圣战？甚至还 结过婚？

我很喜欢这位 艾本祖尔大师 所触发的词汇 肮脏科学 。
